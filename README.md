# 🪵 Parkettleger-Apps (Suite)

Willkommen im offiziellen Repository für spezialisierte Fach-Apps im Parkettleger-Handwerk. Diese Sammlung bietet digitale Unterstützung für die Ausbildung, die Meisterschule und den täglichen Baustellenbetrieb.

---

## 📐 1. ParkettRiss (v1.0)
**Fokus:** Geometrisches Fachzeichnen und präzises Anreißen.
* **Status:** ✅ Fertiggestellt
* **Features:** Interaktive Mittelsenkrechte, Fischgrät-Einteilung, technisches Schriftfeld.
* **Downloads:**
  * [⬇️ Windows-Version (.zip)](https://github.com/BSLehrer/Parkettleger-Apps/releases/latest)
  * [📲 Android-App (.apk)](https://github.com/BSLehrer/Parkettleger-Apps/releases/latest)

## 🛠 Installation & Hilfe
1. **Windows:** ZIP-Ordner entladen und die `parkettriss.exe` starten.
2. **Android:** Die APK auf das Handy übertragen. Eventuell muss die Installation aus "unbekannten Quellen" in den Einstellungen erlaubt werden.

# 🪵 Parkettleger-Apps (Suite)

Willkommen im offiziellen Repository für spezialisierte Fach-Apps im Parkettleger-Handwerk. Diese Sammlung bietet digitale Unterstützung für die Ausbildung, die Meisterschule und den täglichen Baustellenbetrieb.

---

## 📐 1. Geometrische Grundkonstruktionen für Parkettleger (v1.0)
**Fokus:** Fachzeichenen - geometrische Grundkonstruktionen.
* **Features:** schrittweise Darstellung der Konstruktionsschritte.
* **Downloads:**
  * [⬇️ Windows-Version (.zip)](https://github.com/BSLehrer/Parkettleger-Apps/releases/download/v2/windows.zip)
  * [📲 Android-App (.apk)](https://github.com/BSLehrer/Parkettleger-Apps/releases/download/v2/Android.zip)

**Fokus:** Digitales Nachschlagewerk für Holzarten und Normen.

## 🛠 Installation & Hilfe
1. **Windows:** ZIP-Ordner entpacken und die `parkettriss.exe` starten.
2. **Android:** Die APK auf das Handy übertragen. In den Einstellungen muss ggf. die Installation aus "unbekannten Quellen" erlaubt werden.
